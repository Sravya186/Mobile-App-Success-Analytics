# Mobile App Success Analytics Using Python

## Overview

This project analyzes the Google Play Store dataset to understand the factors that contribute to mobile app success.

## Objectives

- Analyze app categories
- Study app ratings
- Compare free and paid applications
- Analyze installs and reviews
- Identify top-rated and most-reviewed applications

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset

Google Play Store Dataset

## Visualizations

- Top App Categories
- Rating Distribution
- Free vs Paid Apps
- Top Categories by Installs
- Reviews vs Installs
- Average Ratings by Category
- Most Reviewed Applications

## Key Insights

- Family and Game categories dominate the Play Store.
- Most applications are free.
- Higher reviews generally correspond to higher installs.
- Some categories consistently receive higher average ratings.
